# No False Claims Rule

## Objective
Enforce the public claims policy inside the code repo.

## Rules
- Reject any code strings or visual elements asserting direct recruitment or employment at high-profile firms (e.g. xAI, Tesla, SpaceX, Google, OpenAI).
- Enforce mandatory project labels:
  - Scottish Accessibility: Academic Dissertation Project
  - SME Validation: Portfolio Project
  - Theatre BI: MSc / Academic BI Project
  - VCCF: Independent R&D Project
  - BA / KPMG: Virtual Simulations
- Publicly block H3+ or Q20+ references.
